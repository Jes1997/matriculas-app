## 🎯 Problema que resuelve

En entornos de garita o control de accesos:

- ❌ Uso de papel y bolígrafo  
- ❌ Búsqueda lenta manual  
- ❌ Riesgo de errores  

👉 Esta app permite:

- ✔ Buscar matrículas en segundos  
- ✔ Registrar camiones rápidamente  
- ✔ Evitar errores humanos  

---

## ✨ Características

- 🔍 Búsqueda en tiempo real  
- ⚡ Entrada rápida con ENTER  
- ➕ Añadir / editar / eliminar  
- 💾 Persistencia con LocalStorage  
- 📤 Exportar datos (JSON)  
- 📥 Importar datos  
- 🌙 Modo oscuro automático  
- 📱 Instalable como app (PWA)  
- 🔌 Funciona sin conexión  

---

## 🧠 Decisiones técnicas

- Sin backend → simplicidad y velocidad  
- LocalStorage → persistencia inmediata  
- PWA → uso como app real  
- UI optimizada para móvil → uso en garita
